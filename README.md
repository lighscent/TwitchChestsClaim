# TwitchChestsClaim

This project aims to automate the collection of bonus points (chests) on Twitch, thereby facilitating interaction on the platform without requiring manual clicks.

## Features

- **Automated Claiming:** Automatically clicks the "Claim Bonus" button when it appears on a Twitch stream.
- **Chests Counter:** Keeps track of how many chests you've successfully claimed.
- **Popup Stats:** Click on the extension icon to see your total claimed chests.

## Installation

1. Clone or download this repository to your local machine.
2. Open your Chromium-based browser (Chrome, Edge, Brave, etc.) and go to the extensions page (e.g., `chrome://extensions/`).
3. Enable **Developer mode** in the top right corner.
4. Click on **Load unpacked** and select the folder `TwitchChestsClaim`.
5. Pin the extension and go watch a Twitch stream. The extension will run in the background and claim chests for you!
